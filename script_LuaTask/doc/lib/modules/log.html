openTrace (v, uartid, baudrate) 
开启或者关闭print的打印输出功能

Parameters:

  • v bool 
    false或nil为关闭，其余为开启
  • uartid 
    输出Luatrace的端口：nil表示host口，1表示uart1,2表示uart2
  • baudrate number 
    number类型，uartid不为nil时，此参数才有意义，表示波特率，默认115200 \
 支持1200,2400,4800,9600,10400,14400,19200,28800,38400,57600,76800,115200,230400,460800,576000,921600,1152000,4000000

Returns:

  • nil

Usage:

  • sys.openTrace(1,nil,921600)
