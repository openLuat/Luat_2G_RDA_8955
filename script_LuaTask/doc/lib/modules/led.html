blinkPwm (ledPin, light, dark)

    ledPin(1)用pins.setup注册返回的方法

     light-亮灯时间ms
  • dark number 
     dark-灭灯时间ms

  • nil

  • led.blinkPwm(lenPin,500,500)

levelLed (ledPin, bl, bd, cnt, gap) 
等级指示灯

Parameters:

  • ledPin function 
     ledPin(1)用pins.setup注册返回的方法
  • bl number 
    亮灯时间ms
  • bd number 
    灭灯时间ms
  • cnt number 
    重复次数 (等级的级别,亮灭1次算数字1)
  • gap number 
    间隔时间 (每次循环周期的间隔)

Returns:

  • nil

Usage:

  • led.leveled(ledPin,200,200,4,1000)
  • 调用函数需要使用任务支持
