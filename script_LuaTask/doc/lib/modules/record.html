Functions

start (seconds, cb) 
开始录音

Parameters:

  • seconds 
     录音时长，单位：秒
  • cb 
     录音结果回调

Returns:

  • result true - 开始录音 其他 - 失败

Usage:

  • result = record.start()
stop ([cbFnc=nil]) 
停止录音

Parameters:

  • cbFnc function [此参数可选，默认值为： nil] 
    停止录音的回调函数(停止结果通过此函数通知用户)，回调函数的调用形式为：
      cbFnc(result)
      result：number类型
              0表示停止成功
              1表示之前已经发送了停止动作，请耐心等待停止结果的回调

Usage:

  • record.stop(cb)
